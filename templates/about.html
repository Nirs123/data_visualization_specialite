    Le projet

    Ce projet de Data Visualisations a commencé en Janvier 2023 lorsque j'explorais les jeux de données sur le site data.gouv.fr.

    Etant élève au lycée l'année précédente, je me suis alors penché vers les jeux de données que proposais le Ministère de l'Education Nationale et je suis alors tombé sur un jeu de données qui répertoriait toutes les infos sur les spécialités enseignés au Lycée: nombre d'étudiant en première, terminale, établissements...

    Avec ce jeu de données, j'ai codé en Python un programme en 2 parties:

    1. Récupération des données voulues depuis les tableaux du jeu de données grâce a plusieurs librairies Python comme Pandas ou Numpy
    2. Génération des graphiques grâces aux données que j'ai extraites et aux librairies comme Matplotlib ou Geopandas.
    Après cela, je me suis dit qu'il était intéressant d'avoir un site internet qui permettait de génerer autant de graphique que voulu, c'est pour cela que j'ai réalisé ce site web avec Python grâce a la librairie Flask.
